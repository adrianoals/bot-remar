# Scripts Manuais

Esta pasta contem utilitarios manuais de operacao/simulacao.

## Arquivos

- `interactive_chat.py`: simulador interativo local com mocks.
- `test_supabase_connection.py`: smoke check manual de conexao com Supabase.

## Observacao

A suite automatizada de testes fica em `tests/`, separada por tipo:

- `tests/unit`
- `tests/contracts`
- `tests/integration`

Use `scripts/` quando quiser validar algo manualmente fora da suite automatizada.
