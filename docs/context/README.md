# Referencia n8n

O arquivo `ChatRemar.json` e a referencia de fluxo original usada para comparar comportamentos da versao Python.

## Uso recomendado

- Usar para validar textos, transicoes de estado e mapeamentos de integracao.
- Nao editar esse arquivo manualmente sem necessidade.
- Toda evolucao da versao Python deve manter compatibilidade funcional com regras ativas de negocio.

## Observacao

A versao Python pode divergir do n8n quando houver decisao explicita de produto/operacao (ex.: remover etapa de estado geografico).
